## Week 1 Focus: AI Workflows Mastery

This week focuses on mastering AI collaboration workflows through theory and hands-on research practice. Candidates will learn core mindset, critical thinking principles, and three essential workflows (Layered Questioning, Solution Exploration, Iterative Refinement) before applying them to research Test-Driven Development (TDD) and testing mindset.

## Homework Assignment

**Research Task:** Research Test-Driven Development for building reliable CLI tools with AI assistance

**Research focus:**
- **Core principles** - What is TDD? What are Red, Green, and Refactor?
- **Testing levels** - Difference between unit tests, integration tests, and end-to-end tests
- **AI validation** - How tests help verify and improve AI-generated code
- **CLI testing** - What should be tested in a CLI tool: commands, validation, file storage, and errors
- **Common mistakes** - Over-testing, weak assertions, testing implementation details, and blindly trusting AI output

**Apply workflows:**
1. **Layered Questioning** - Research → Brief → Example → Validation
2. **Solution Exploration** - Explore options → Compare pros/cons → Choose with context
3. **Iterative Refinement** - Review AI suggestions → Refine → Feedback → Validate

**Deliverable:** Research documents are saved with:
- Core principles of TDD and Red-Green-Refactor
- Comparison of unit tests, integration tests, and end-to-end tests
- Examples of tests for a Ticket Manager CLI
- How testing helps control AI-generated implementation
- Common testing mistakes and how to avoid them
- Evidence of applying all 3 workflows

## Acceptance Criteria

- [ ] Research content is documented: TDD principles, testing levels, CLI test examples, AI validation
- [ ] Research process with AI is tracked: Workflows applied and iterations documented
- [ ] Research findings can be explained clearly when submitting
- [ ] Questions about TDD, test types, and AI-generated code validation can be answered based on research

## Preparation for Week 2

After Week 1, you should be ready to:
